# Eight-queens-puzzle
> spring 2014

# Description
Using a regular chess board, the challenge is to place eight queens on the board such that no queen is attacking any of the others.

# About this project
In this program, we used the following algorithms:
* Hill-climbing
* simulated annealing
* Genetic algorithms
